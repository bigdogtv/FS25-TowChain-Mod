Tow Chain — Rusty (Medium, Auto-Attach)

Files:
- modDesc.xml
- towChain.xml
- towChain.i3d
- autoAttach.lua
- chain_diffuse.png (convert to .dds for best visuals)
- icon.png
- README.txt

Install:
1. Upload the folder to your GitHub repository.
2. Download as ZIP or clone the repository.
3. Place the 'FS25-TowChain-Mod' folder into:
   Documents\My Games\FarmingSimulator25\mods\
4. Launch FS25 → Shop → Miscellaneous → Rusty Tow Chain (Medium)

Notes:
- Replace placeholder chain links in Giants Editor with proper meshes for best visuals.
- Auto attach works on PC; adjust attach distance in autoAttach.lua if needed.
